Pythonの魅力

Pythonは多くあるプログラミング言語のうち、人気の高い言語の一つです。 プログラミング初心者でもかなり手軽に始められ、簡単なプログラムを書くことができます。 また、豊富なライブラリがあり、それらを使うことで様々なことを行うことができます。

例えば、YoutubeやDropBoxはPythonで開発されたサービスです。 さらには、ビッグデータや機械学習などもPythonの得意分野です(偏見)。

Pythonコードの例

ではようやくPythonの説明に入っていきます。
まずは、この講習で使うプログラムを一度見せます。 次のコードは、永遠にじゃんけんをし続けるソースコードです(Lesson.pyにも同じ内容があります)。

# -*- coding: utf-8 -*- import random while True: print(" [0]グー") print(" [1]チョキ") print(" [2]パー") player = int(input("あなたの手は？")) if player == 0: print("あなたはグーを出しました") elif player == 1: print("あなたはチョキを出しました") elif player == 2: print("あなたはパーを出しました") else: print("0~2の数字を入力してください") continue com = random.randint(0,2) if com == 0: print("コンピュータはグーを出しました") elif com == 1: print("コンピュータはチョキを出しました") elif com == 2: print("コンピュータはパーを出しました") com += 3 if (com - player) % 3 == 1: print("あなたの勝ちです") elif (com - player) % 3 == 2: print("あなたの負けです") else: print("あいこです") print("もう一度？") print(" [0]はい") print(" [1]いいえ") again = int(input(":")) if again == 0: print("じゃんけんをします") else: print("じゃんけんをしません") break

これを、Lesson.pyに記述して、下のターミナルに以下コマンドを入力して実行してみましょう。

python Lesson.py

出力(表示)

学習の前に、先ほどのファイルとは別に新しいファイルを作りましょう。
ここでは「Learn.py」という名前で保存します。
これに、色々なコードを記述し、最終的にはLesson.pyと同じようなコードを書きます。

では説明に入ります。
まずは、プログラミングのはじめの一歩、出力を行うコードを書きます。
Pythonでは、「print」という関数を使います。
では実際にコードを見てみましょう。

# -*- coding: utf-8 -*- print("[0]グー") print("[1]チョキ") print("[2]パー") 実行結果 [0]グー [1]チョキ [2]パー

これで、出力ができるようになりました。

ちなみに、一番上のものはpythonに「これはUTF-8で書かれているよ〜」ということを教えるためのものです。おまじない程度に覚えておくといいでしょう。
また、「#」以降はコメントとして読み込まれます。コメントはプログラムには一切関係しません

入力を受け取る

次に、入力を受け取ってみましょう。 これは、「input」という関数を使います。 この関数は、文字列を受け取ります。 ここでは、「あなたの手は？」という文字列を出力します。 そして、「input」関数を使って、入力を受け取ります。

また、入力には、それの受け皿となるものが必要です。 プログラミングでは、そのようなものを「変数」といいます。 ここでは、「player」という変数に入力を代入します。 これにより、「player」には、入力された数字が代入されます。 これを、プログラムの中で使うようにします。

では、プログラムを書いていきましょう。 先ほどのプログラムに追記する形で書きます。

# 出力のプログラム(省略) # 入力を受け取る player = int(input("あなたの手は？"))　# "あなたの手は？"は省略可能 print(player) 実行結果(1と入力) あなたの手は？1 1

ソースコードでは、input関数をint()で囲んでいます。こうすることで、文字列だった入力を整数に変換しています。(こうすることで後の処理が楽になります)
これで、入力が受け取れるようになりました。

条件で分岐させる

次は、受け取った数値に応じてグー、チョキ、パーを判別できるようにしましょう。
このような処理を条件分岐といい、Pythonではif文というものを使います。ここから少し複雑になって行きますが、頑張って行きましょう。
Pythonでは、if文は次のように書きます。

if 条件式1 : 条件1を満たす場合の処理 elif 条件式2 : 条件式1は満たさないが、条件式2は満たす場合の処理 else: どの条件式も満たさない場合

条件式は、簡単に説明すると「2つの値を比較した結果」です(例外は多いですが、基本的な意味はこれです)。つまり、同値か、大きいか小さいか、といったものです。
基本的なものは6種類あり、それぞれ以下のように書きます。

条件式 意味
x == y xとyが同じ値かどうか
x != y xとyが違う値かどうか
x > y xがyより大きいかどうか
x < y xがyより小さいかどうか
x >= y xがy以上かどうか
x <= y xがy以下かどうか

また、if文の「elif」の部分は0個以上書くことができます。
では、if文を使って受け取った数字がどれに該当するかを判別しましょう。

# 出力のプログラム(省略) # 入力のプログラム(省略) if player == 0: print("グー") elif player == 1: print("チョキ") elif player == 2: print("パー") else: print("0~2を入力してください") 実行結果(1と入力) チョキ

これで、条件分岐ができました。

次に、コンピュータ側の手も決めましょう。まず、乱数生成器を使うため、下のように追記します。

# -*- coding: utf-8 -*- import random # ...

こうすることで、乱数が使えるようになりました。あとはプレイヤー入力の部分を少し変えて、あとは先ほどと同じです。
コンピュータの手はcomという変数に入れておきます

# 入力を受け取る(プレイヤー) com = random.randint(0,2) if com == 0: print("コンピュータはグーを出しました") elif com == 1: print("コンピュータはチョキを出しました") elif com == 2: print("コンピュータはパーを出しました")

これで、コンピュータがグー、チョキ、パーのいずれかを出しています。
これで、どちらが勝ったかを判別しましょう。

計算をする

さて、ジャンケンの勝ち負けはif文で9通り全てを検証する方法もありますが、今回は少し頭の良い方法を使います。
コンピュータの手の数値に3を足して、プレイヤーの手の数値との差を取り、それを3で割った余りで判別します(1なら勝ち、2なら負け)。

Pythonでの計算は、「x + y」という形で行うことができます。
この中の「+」を演算子といい、代表的なものには以下のものがあります。

演算子 意味
+ 和
- 差
* 積
/ 商
% 剰余(あまり)

では、これらを使って、演算を行い、勝ち負けの判定をしていきましょう。

# 出力(省略) # 入力(プレイヤー) # コンピュータの手を決める com += 3 # 特殊!(複合演算子といい、演算と代入を同時に行います) if (com - player) % 3 == 1: print("あなたの勝ちです") elif (com - player) % 3 == 2: print("あなたの負けです") else: print("あいこです")

これで、演算ができるようになりました。

ループさせる

ここまでのプログラムでも、じゃんけんとしては十分に機能しますが、全体の処理をループさせて、複数回のゲームを行うこともできます。
これを「繰り返し」といい、「while文」というものを使います。次のような感じです。

while 条件式: ループさせる処理

このループは、条件式が満たされる限り、ループさせる処理を繰り返します。
ここでは、この条件式は「True」として無限ループさせて、プレイヤーの操作の後に「break」という文によって強制的にループから出させます。

さて、ここでwhile文を書くとなると、今まで書いたコードの全ての先頭にタブを入れる必要があります、これは面倒なので、以下のソースコードをコピーしてください。 (今までのコードのまとめです)

# -*- coding: utf-8 -*- import random while True: # 無限ループさせる print(" [0]グー") print(" [1]チョキ") print(" [2]パー") player = int(input("あなたの手は？")) if player == 0: print("あなたはグーを出しました") elif player == 1: print("あなたはチョキを出しました") elif player == 2: print("あなたはパーを出しました") else: print("0~2の数字を入力してください") continue com = random.randint(0,2) if com == 0: print("コンピュータはグーを出しました") elif com == 1: print("コンピュータはチョキを出しました") elif com == 2: print("コンピュータはパーを出しました") com += 3 if (com - player) % 3 == 1: print("あなたの勝ちです") elif (com - player) % 3 == 2: print("あなたの負けです") else: print("あいこです")

これで、ループを使って、複数回のゲームを行うことができます。
他にも、ループさせる方法として、「for文」というものがあります。興味のある人は調べてみると良いでしょう(こちらの方が良く使われます)。

さて、これで、Pythonの基本的な処理はかけるようになりました。しかし、まだこのじゃんけんゲームには足りないものがあります。ゲーム終了判定です。
最後に、この講座の最後に、「じゃんけんゲームを終了する」という文を追加していきます。これは自分でやってみましょう。

ヒント:
ループを強制終了させるには、「break」と書いてあげると良いでしょう。
